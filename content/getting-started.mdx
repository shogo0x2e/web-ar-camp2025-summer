# Getting Started

Web ARの開発を始めるためのガイドです。

## 環境構築

1. Node.js (v18以上) をインストール
2. プロジェクトをクローン
3. 依存関係をインストール

```bash
npm install
```

## 最初のARアプリ

簡単なARアプリケーションを作成してみましょう。

```javascript
// 基本的なARシーンの設定
const scene = new ARScene();
scene.init();
```
