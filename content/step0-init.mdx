# Step 0: プロジェクト初期化

Next.jsプロジェクトを0から作成してWebARアプリの基礎を構築します。

## 実装手順

### 1. プロジェクト作成

```bash
npx create-next-app@latest web-ar-project
```

### 2. 設定選択

作成時のオプションを以下のように選択してください：

- ✅ TypeScript
- ✅ ESLint  
- ✅ Tailwind CSS
- ✅ src/ directory
- ✅ App Router
- ❌ Turbopack (for compatibility)

### 3. 開発サーバー起動

```bash
cd web-ar-project
npm run dev
```

## 期待される結果

- Next.jsプロジェクトが正常に作成される
- 開発サーバーが `http://localhost:3000` で起動する
- ブラウザでNext.jsのデフォルトページが表示される
