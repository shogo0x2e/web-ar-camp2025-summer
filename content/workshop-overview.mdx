# WebAR Workshop Overview

Next.js + React Three Fiber + MindARを使ったWebARアプリケーション開発ワークショップです。

## ワークショップ概要

### 目標
2日間で「名刺上に3Dオブジェクトをトラッキング表示するWebARアプリ」を作成します。

### 想定参加者
- React/Next.jsの基本知識がある方
- WebGL/Three.jsは未経験でもOK
- ARに興味がある開発者

### 最終成果物
- 画像マーカー上に3Dオブジェクトを表示
- パーティクルエフェクト
- ポートフォリオ/SNSリンクボタン
- レスポンシブ対応
- Vercel等でのデプロイ

## 技術スタック

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **AR**: MindAR (Image Tracking)
- **3D**: Three.js + React Three Fiber
- **デプロイ**: Vercel

## 必要な準備

### 開発環境
- Node.js 18+ インストール済み
- VS Code (推奨エディター)
- モダンブラウザ（Chrome/Safari推奨）
- スマートフォン（実機テスト用）

### 素材準備（オプション）
- ロゴ画像 (.png, 512x512px推奨)
- 3Dモデル (.glb/.gltf形式)
- ポートフォリオURL

### アカウント
- GitHub（コード管理用）
- Vercel（デプロイ用）

## 学習の流れ

各ステップは段階的に積み上がるよう設計されています：

1. **基礎固め**: React/Next.jsの確認
2. **AR導入**: カメラとトラッキング
3. **3D表示**: オブジェクトの配置
4. **エフェクト**: 視覚的魅力の向上
5. **UI統合**: ユーザビリティの完成

## トラブルシューティング

### よくある問題
- カメラ許可が取れない → HTTPS/localhost確認
- ライブラリが読み込まれない → CDNのバージョン確認
- 3Dが表示されない → コンソールエラーチェック
- パフォーマンス問題 → デバイス性能に応じた調整
